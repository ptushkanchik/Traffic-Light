Программа представляет собой реализацию работы дорожного светофора. Функции реализованы на прототипном наследовании. Для запуска работы программы - введите время работы красного и зеленого фонарей в секундах (от 4 до 999) и нажмите "СТАРТ".В данный момент обработчик событий кнопки "СТОП" не активен и находится в разработке.
